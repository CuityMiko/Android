

## 快捷键

### 编辑

| 快捷键 | 作用 | 备注 |
|:-------------|:-------------|:-----|
| Shift+Enter | 在当前行下面新增一行然后跳至该行 |  |
| Ctrl+Enter | 在当前行上面新增一行然后跳至该行 | 此功能光标在首行时才有效 |
| Ctrl+Y | 删除当前行 |  |
| Ctrl+Alt+Space | 类名自动完成（自动补齐） | s |
| Ctrl+Alt+L | 代码格式化 | 和QQ的快捷键冲突 |
| Alt+Enter | 自动修正 |  |
| **Ctrl+Shift+C** | **复制当前文件的路径** | 这个快捷键有点叼啊 |
| Ctrl+Shift+V | 选最近使用的剪贴板内容并插入 |  |
| Ctrl+/ | 注释 // |  |
| Ctrl+Shift+/ | 注释 `/*...*/` |  |
| F2/Shift+F2 | 跳转到下/上一个错误语句处 | s |
